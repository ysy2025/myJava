属性/成员变量

注意事项和细节说明
属性的定义语法同变量:
属性的定义类型可以是任意类型,包含了值类型或者引用类型

scala中声明一个属性,必须显式初始化,然后根据初始化数据类型,自动腿短;属性类型可以省略
如果赋值为null,一定要加类型,因为不加,属性的类型就是Null类型了

定义属性的时候可以暂时不赋值,这样需要使用符号_,让系统分配默认值.
不同对象的属性是独立的,互不影响.



如何创建对象
如果不希望改变对象的引用(内存地址),应该声明为val;否则为var;scala推荐val


继承和多态
(1)子类继承父类的属性和方法
(2)scala 是单继承


多态
多种状态
一种接口可以有多种实现方式--多态
java 静态绑定属性(跟类走);动态绑定方法(跟对象走)
scala 动态绑定属性(跟对象走);动态绑定方法(跟对象走)

伴生对象
单例对象
(1)单例对象采用 object 关键字声明
(2)单例对象对应的类称之为伴生类，伴生对象的名称应该和伴生类名一致。
(3)单例对象中的属性和方法都可以通过伴生对象名(类名)直接调用访问。
